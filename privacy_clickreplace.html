Click&Replace Privacy Policy

Parce que cliquer, c'est déjà un effort.

Click&Replace does not collect, transmit, or share any personal data. All processing is performed locally on the user's computer.

1. Data Collection

The application does not collect or transmit personal information, analytics data, identifiers, or usage statistics.

All DWG processing, replacement operations, scans, backups, and report generation are executed locally on the user's machine.

2. Local File Processing

Click&Replace processes user-selected DWG and XLSX files locally in order to perform text replacement operations.

The application may temporarily create:

  • Backup files
  • Scan results
  • Replacement reports
  • Temporary processing files

These files remain on the user's computer and are never transmitted externally.

3. Data Sharing

No data is shared with third parties. The application does not use:

  • Cloud services
  • Analytics platforms
  • Advertising services
  • Telemetry systems
  • External databases
  • Third-party tracking SDKs

4. Data Retention

Click&Replace does not retain personal data.

Temporary processing files created during execution are automatically cleaned whenever possible. Backup files and generated reports are intentionally preserved locally so users can review or restore their work if necessary.

5. User Rights

Because no personal data is collected, transmitted, or stored externally, there is no personal user database to access, modify, revoke, or delete.

6. Offline Operation

Click&Replace operates entirely offline for all DWG processing operations.

An internet connection is only used when the user explicitly clicks the purchase link, which opens an external store page in the user's default web browser.
