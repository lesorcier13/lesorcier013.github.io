Privacy Policy — Click&PDF

Click&PDF does not collect, store, transmit, or share any personal data. All processing is performed locally on the user's computer.

1. Data Collection

The application does not collect any personal information, usage data, file contents, file paths, identifiers, or analytics data.

2. Data Sharing

No data is shared with third parties. No analytics services, advertising platforms, cloud services, or third-party SDKs are used.

3. Data Retention

No personal data is retained by the application. Temporary files created during processing (DSD files, SCR scripts, template copies) are deleted automatically at the end of each operation.

4. DWG File Processing

DWG files are processed locally using AutoCAD components installed on the user's system. Files are opened in read mode only for PDF generation and are never transmitted outside the local machine.

When the local cache option is enabled, DWG files are temporarily copied to a local folder on the user's computer to improve performance. These copies are automatically deleted at the end of the session.

5. PDF Output

Generated PDF files are saved locally in the source folder of the processed drawings. No PDF content is transmitted or shared externally.

6. License Registry

The application stores license information (activation status and trial counter) in the Windows registry under the current user's profile only (HKEY_CURRENT_USER). This data is local, never transmitted, and can be removed by uninstalling the application.

7. User Rights

Because no personal data is collected or stored externally, there is no user data to access, modify, revoke, or delete.
